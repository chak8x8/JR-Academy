Explain: every CSS thing you used

@import url(...) — load a web font

Downloads the Roboto font from Google so your text uses that font instead of the browser’s default.

body { font-family: "Roboto", sans-serif; }

Sets the default font for the whole page. Everything inside the body inherits this font unless overridden.

h1 { color; display: flex; justify-content: center; }
  • color — the text color for the heading.
  • display: flex — makes the h1 act like a flex container for its inline content.
  • justify-content: center — centers that content horizontally.
.container { display: flex; justify-content: center; }

Makes the container a flexbox and centers the inner .wrapper horizontally.

.wrapper { display: flex; flex-wrap: wrap; }
  • display: flex — puts the cards in a row.
  • flex-wrap: wrap — if there isn’t enough space in the row, cards flow down to the next line.
.card box styles
  • max-width: 300px — don’t let cards get wider than 300px.
  • min-height: 250px — keep cards at least 250px tall.
  • background-color — card color (you change it on cards 2 and 4).
  • box-sizing: border-box — width/height include padding & border (easier sizing).
  • padding — inner space inside the card.
  • border-radius — rounded corners.
  • margin — space around the outside of the card.
  • box-shadow — drop shadow for depth.
  • transition + transform: translateY(-8px) on .card:hover — smooth “lift up” effect.

Tip: Your original had transition: all 0.3 ease; — it needs seconds: 0.3s.

.card_image img { width: 100%; }

Makes the image fill the card’s content width. If images have different shapes, you can add:

.card_image img {
  width: 100%;
  height: 180px;       /* pick a height */
  object-fit: cover;   /* crop nicely */
  border-radius: 8px;
}

Or use aspect-ratio: 16 / 9; on a wrapper to keep all tiles the same shape.

.card:nth-child(2) and .card:last-child

These are structural selectors:

  • :nth-child(2) → selects the 2nd card.
  • :last-child → selects the last card.

You used them to change background colors of those specific cards.

.card_info { display:flex; justify-content: space-between; align-items: center; }

Places the time text on the left and the button on the right, on the same line, vertically centered.

.card_price button styles
  • background-color, color — button colors.
  • border: none — remove the default border.
  • border-radius: 20px — pill-shaped button.
  • padding: 5px 20px — inner space.
  • cursor: pointer — shows hand cursor on hover.
When would I use grid instead of flex?

Flex is best for one direction (a row OR a column). Grid is best when you need control over both rows AND columns (a 2D layout). Try the “Make wrapper use grid” toggle above to compare.
